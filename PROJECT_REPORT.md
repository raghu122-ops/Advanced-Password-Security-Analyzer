### 1. Project Objective
The goal is to provide a tool that helps users identify weak or breached passwords to prevent unauthorized access and improve overall digital security.

### 2. Cybersecurity Relevance
This project addresses Identity and Access Management (IAM) by ensuring users create strong passwords and identifying compromised credentials using global breach databases.

### 3. Implementation Details
* Developed a Flask-based backend to handle security logic.
* Implemented SHA-1 hashing to securely query the HaveIBeenPwned API.
* Used Regular Expressions (Regex) for character pattern matching and strength scoring.
* Calculated Password Entropy to provide a mathematical measure of security.
